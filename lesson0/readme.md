## Setup

Virtual environment

```bash
conda create --name hello-world-geometry python=3.11
conda activate hello-world-geometry
```

Install dependency

```bash
pip install -r requirements.txt
```

## Run App

```bash
python app.py
```

## Syntax

```bash
conda env list

conda activate hello-world-geometry
conda deactivate

conda create --name hello-world-geometry
conda env remove --name hello-world-geometry
```
